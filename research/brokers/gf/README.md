# 广发策略研报库

本目录用于整理公开可访问的广发证券策略研报资料，供知识库检索与后续量化研究参考。

## 目录建议

- `index.md`：总索引
- `metadata\reports.json`：结构化元数据
- `macro\`：宏观/总量/资产配置
- `a_share\`：A股市场策略
- `industry_style\`：行业配置/风格轮动
- `weekly_monthly\`：周报/月报/年度策略
- `source_links\`：来源与链接留档

## 原则

- 优先保留公开可访问来源
- 先做索引与元数据整理，再扩展文件落地
- 避免收录明显无关、失效或来源不清的内容
